# Data-tool

## 介绍 📘

​ Data-tool-end 是一款基于**FastAPI**、**PANDAS** 开发的数据分析后端
## 项目预览 👀



## 代码仓库 🌟

- GitHub：https://github.com/L4mb3stone/data-tools-end

## 快速开始 🚀

- **拉取项目** (您需要先安装 Git)

```bash
# GitHub
git pull https://github.com/L4mb3stone/data-tools-end.git
```

- **运行项目**


```bash
cd 拉取项目目录
pip install pyinstaller
pip install pandas
pip install FastAPI
pip install pymysql
```
